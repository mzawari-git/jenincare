1. Upgrading the Ensemble Logic (Outlier Rejection)
Moving to a weighted median with outlier rejection will drastically stabilize your scores. If Advanced hallucinates an 85 but CV and Basic sit at 30 and 35, the median filters the noise perfectly.

Here is the conceptual Kotlin approach for EnsembleAnalysisEngine.kt:

Kotlin
private fun calculateRobustScore(
    advScore: Float, advWeight: Float,
    cvScore: Float, cvWeight: Float,
    basicScore: Float, basicWeight: Float
): Float {
    val engineData = listOf(
        EngineResult(advScore, advWeight),
        EngineResult(cvScore, cvWeight),
        EngineResult(basicScore, basicWeight)
    ).sortedBy { it.score }

    // 1. Calculate Median (middle value of 3)
    val medianScore = engineData[1].score

    // 2. Outlier Rejection & Weight Adjustment
    var totalWeight = 0f
    var weightedSum = 0f

    for (result in engineData) {
        val deviation = Math.abs(result.score - medianScore)
        // If deviation > 20 points, penalize weight by 50%
        val finalWeight = if (deviation > 20f) result.weight * 0.5f else result.weight
        
        weightedSum += (result.score * finalWeight)
        totalWeight += finalWeight
    }

    return if (totalWeight > 0) weightedSum / totalWeight else medianScore
}
